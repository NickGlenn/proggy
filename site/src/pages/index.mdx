## Usage

```sh
# reads PROGRESS.md in the current directory
proggy

# specify a file
proggy path/to/TODO.md

# live-watch mode
proggy --watch

# print version
proggy --version
```

## Markdown format

proggy expects `##` headings for sections and `- [x]` / `- [ ]` checkboxes for items.

```md
## Backend

- [x] VM interpreter
- [x] C codegen
- [ ] LLVM codegen

### Subsections group under the parent

- [x] Register allocation
- [ ] Instruction selection
```

- `## Heading` starts a new section row in the table
- `### Subheading` groups items under the current section
- `- [x]` counts as done, `- [ ]` counts as remaining

## Install

### Homebrew (macOS / Linux)

```sh
brew install nickglenn/tap/proggy
```

### Scoop (Windows)

```powershell
scoop bucket add nickglenn https://github.com/nickglenn/scoop-bucket
scoop install proggy
```

### Go

```sh
go install github.com/nickglenn/proggy@latest
```

### From source

```sh
git clone https://github.com/nickglenn/proggy.git
cd proggy
go build -o proggy .
```

### Debian / Ubuntu

Download the `.deb` from the [latest release](https://github.com/nickglenn/proggy/releases) and install:

```sh
sudo dpkg -i proggy_*.deb
```
